How Our AI Matching Engine Works

Bell24h's AI Matching Engine uses sophisticated algorithms and machine learning to create perfect matches between buyers and suppliers.

Natural Language Processing

Our advanced NLP algorithms analyze RFQ descriptions to understand specific requirements, technical specifications, and desired outcomes. This allows us to go beyond basic keyword matching.

Key Components:

  • Text classification and categorization of RFQs
  • Entity recognition for technical specifications
  • Sentiment analysis for quality preferences
  • Contextual understanding of industry-specific terminology

SHAP Explainability

Our AI provides transparent explanations for every match using SHAP (SHapley Additive exPlanations) values. This explainable AI approach helps buyers understand exactly why a supplier is recommended.

Benefits of SHAP:

  • Completely transparent scoring methodology
  • Quantitative contribution of each factor to the match score
  • Identifies both positive and negative contributing factors
  • Builds trust in AI-driven decision making

Vector Embeddings & Similarity Matching

We use advanced vector embeddings to represent suppliers and RFQs in a high-dimensional space. This allows us to calculate similarity scores that capture complex relationships beyond simple keyword matching.

Technical Features:

  • 768-dimensional vector embeddings for rich representation
  • Cosine similarity for effective semantic matching
  • Optimized with Supabase PGVector for fast retrieval
  • Continuous learning from user feedback and transactions
